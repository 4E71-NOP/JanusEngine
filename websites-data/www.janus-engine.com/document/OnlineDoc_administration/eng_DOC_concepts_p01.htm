Users

The user managment is based on a hierarchy. Each user is a member of one or several groups. The group will determine what users will see and what users will be able to do.

Groups

The groups are important because they are linked to the menus. One menu only allows one group. If a user is not a member of a designated group by a menu, the associated menu will not be displayed.

There are 3 group types

  • Anonymous : is the default group for any user that is not authenticated.
  • Reader : is the type for registered users.
  • Staff : is the type for registered users that have special access to the website.
