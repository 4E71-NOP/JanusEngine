Decoration

« décoration » : Designate a decoration.

Associated arguments of « decoration »
name Name used as internal reference.
state 5 aviable types: caligraph, 1_div, elegance, exquisite, elysion
directory Defines the working directory.
filter Text field. Enable filtering on the internal name of the target elements for the "show" command line.

Depending on the selected type, the command changes. The last part of the command is mainly difinitions. For the 1_div type the selection can be on all borders. Ex: border_all_width '10', border_left_width '5'

Example:

Blocks

Blocks are part of décoration and allow to define a style for a module (le contenu, le menu, etc.).

« 1_div » : defines the decoration type 1_div.

Type : « 1_div »
background_color Color type argument
border_<selection>_width Size type argument
border_<selection>_color Color type argument
border_<selection>_style Accept following values: solid , none , dotted , dashed , double , groove , ridge , inset , outset
padding_<selection> Size type argument
margin_<selection> Size type argument

"N°" means accepted numbers by the engine for this decoration. The decoration named « elegance » accepts the following numbers : 11, 12, 13, 21, 22, 23, 31,32, 33.
