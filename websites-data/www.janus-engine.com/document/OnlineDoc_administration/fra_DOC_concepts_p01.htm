Les utilisateurs

La gestion des droits utilisateur repose sur un système hiérarchisé. Chaque utilisateur dépend d'un ou plusieurs group. L'appartenance a group va déterminer ce que l'utilisateur voit du site ou peut y faire.

Les groupes

Les groupes sont important car ils sont liés aux catégories. Le catégories n'acceptent qu'un seul group. Si un utilisateur ne fait pas parti d'un group référencé par une catégorie, alors le menu associé à la catégorie ne sera pas affiché.

Il y a trois types de groupes

  • Anonyme : c'est le type par défaut pour tout utilisateur qui arrive sur le site sans être authentifié.
  • Lecteur : C'est le type pour tout utilisateur enregistré.
  • Staff : C'est le type pour tout utilisateur enregistré ayant des droits spéciaux sur le site.

Les catégories

Les catégories sont les éléments qui permettent de forger un menu. Chaque catégorie référence un group qui aura le droit de visualiser le menu défini par elle. Les catégories définissent aussi l'article auquel se rapporte le menu défini.
