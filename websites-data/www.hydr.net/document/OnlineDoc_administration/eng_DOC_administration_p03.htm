Groups managment

The interface enbale the user to access a configuration pannel for managing groups of a website. The pannel only deals with the browsed website groups.

User managment

The interface enbale the user to access a configuration pannel for managing users of a website. The pannel only deals with the browsed website users. The interface enable the user to select users depending on criterias. The criterias are: Group; state (Active/deleted). The user can filter with a string on the login. Ex: « brad » will output login like « BradAnjelina ». The user only have to click on the login in the result framework to get the user details. The interface enables the admin to modify all informations about the user. The user will be able to add a commentary for admin purposes, only viewable by admins, and to modify the access to the website modules.
