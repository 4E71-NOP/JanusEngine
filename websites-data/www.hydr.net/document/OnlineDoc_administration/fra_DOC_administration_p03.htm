Gestion des groupes

L'interface propose un panneau de configuration qui permet de gérer les groupes d'un site donné. Le panneau ne traite QUE des utilisateurs du site en cours de navigation.

Gestion des utilisateurs

L'interface propose un panneau de configuration qui permet de gérer les utilisateurs d'un site donné. Le panneau ne traite QUE des utilisateurs du site en cours de navigation. La page permet de voir une sélection d'utilisateur correspondant a des critère donnés. Ces critères sont : Le group; L'état (Actif/supprimé). Accessoirement l'utilisateur peut choisir en plus les 'login' contenant une chaine de caractère précis. Ex « brad » donnera dans les résultats « BradAnjelina ». Il suffit de cliquer sur le login souhaité dans le cadre des résultats pour obtenir la fiche de l'utilisateur (abonné). La fiche permet de manipuler toutes les informations du l'abonné. En outre, il pourra ajouter un commentaire destiné a l'usage des administrateurs.
